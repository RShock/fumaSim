# skill

所有的技能都应该有一个触发器，一个验证是否触发的开关，和对应的效果

它们被命名为trigger check 和 cast

所有的buff其实也是一种技能

这里收录了3种技能：

* 普通buff(Buff):不可堆叠

* 堆叠buff(uniqueBuff):可堆叠

* 开关buff(switchBuff):有一个额外的开关，当他关闭时虽然buff无法发动，但是依然
显示，并且提示玩家该buff未发动
